# Api Ninja pro

An advanced, high-performance API testing workbench designed with a premium aesthetic.

## Features
- **Dynamic Proxy**: Bypass CORS issues with a built-in Node.js proxy server.
- **Modern UI**: Sharp, professional design with light and dark mode support.
- **Environment Management**: Easily switch between local, staging, and production environments.
- **History & Persistence**: All your requests are saved locally for future use.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server (runs both Vite and the Proxy):
   ```bash
   npm run dev
   ```

## Technology Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Express (Proxy Server)
- Dexie.js (Local Persistence)
